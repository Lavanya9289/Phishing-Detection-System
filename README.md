# 🔐 AI-Powered Phishing Detection System

## 📌 Project Description
This project is a machine learning-based system that detects whether a website URL is safe or phishing.

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- CountVectorizer
- Logistic Regression

## 🚀 Features
- Detects phishing websites
- Uses AI/ML model
- Simple and efficient

## 📊 How it Works
1. Load dataset (phishing.csv)
2. Convert URL into numerical features
3. Train ML model
4. Predict website safety

## ▶️ Run the Project
```bash
python phishing_detection.py

## 🎯 Output
- Safe Website
- Phishing Website Detected
